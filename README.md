# Bank you

![](https://i.imgur.com/KY7zpCp.jpeg)

### Features

- USDC & USDT support (Ethereum);
- Transaction history;
- Balance;
- Totally secure (we don't use your private key);
- Dynamic color with Material You.

### Practices
- MVVM
- Jetpack Compose
- Koin (Dependency injection)
- Ktor
- DataStore
